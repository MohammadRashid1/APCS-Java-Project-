# Student Management System - APCS Semester 1 Project

A JavaFX-based Student Management System with GUI, login system, dashboard, CRUD operations, and JSON data persistence.

## Features
- ✅ User authentication (Login/Logout)
- ✅ Role-based access (Admin/User)
- ✅ Dashboard with statistics and charts
- ✅ Complete CRUD operations for students
- ✅ JSON data persistence
- ✅ Input validation and error handling
- ✅ Search functionality
- ✅ Professional GUI with JavaFX

## Prerequisites
- Java JDK 17 or higher
- JavaFX SDK 17 or higher
- Gson library (for JSON handling)
